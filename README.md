# 💬 Simple Chatbot

A simple chatbot application built using **HTML, CSS, JavaScript, and PHP**.
This project allows users to interact with a virtual assistant and receive responses based on predefined logic.

---

## 🚀 Features

* Interactive chatbot interface
* Real-time user input handling
* Predefined automated responses
* Lightweight and beginner-friendly project structure

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP

---

## 📌 About the Project

This project was initially created as a beginner-level implementation of a chatbot.
Over time, it has been improved to better demonstrate how front-end and back-end technologies can work together to simulate conversational behavior.

It serves as a great starting point for building more advanced AI-powered chatbot applications.
